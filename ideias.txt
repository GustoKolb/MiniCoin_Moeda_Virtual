Temos que fazer um sistema cliente servidor, em que o servidorguarda uma blockchain com transações de moedas chamadas MiniCoin.

A blochchain não parece difícil. Parece uma lista encadeada em que não se retira os nodos e cada novo nodo depende dos anteriores para construir o seu rash. Nesse sentido precisamos:

- Definir a linguagem
- Definir o protocolo da camada de transporte (TCP ou UDP)
- Criar o cliente
- Criar a interface do cliente
- Ver como conectar o cliente com o servidor (se tem que ser feito na mesma máquina ou não)
- escolher hash e aplicá-lo
- Montar lista
- Criar servidor
- Criar sistema de logs do servidor
- Testes a Rodo
- Montar página HTML
- Fazer relatório, de preferência com esquemas visuais e tals.
